## Introduction

Anime Scheduler is an app that schedules your anime according to users' preferences: users can either provide a specific date range or the number of episodes to be watched a day. The app utlizes a [REST API](https://jikan.moe) to obtain data of anime users have requested. Moreover, the anime scheduler app provides a calendar interface the user can take advantage of to view the number of episodes to be watched on a specific day. Lastly, the app displays some basic statistics in order for the users to have a better understanding on how much time they're spending on wathing anime.
